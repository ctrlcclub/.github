# Security Policy

## Reporting a vulnerability
If you discover a security issue in this repository or its community resources, please report it privately to the maintainers.

- Do not open a public issue for security reports.
- Provide steps to reproduce and any relevant context.

## Response expectations
We will acknowledge your report within 7 days and provide a status update as we investigate.

## Supported content
This repository hosts community documents and templates. Security reports are focused on accidental disclosure of private data, unsafe links, or other security risks in content.
