# Governance

This document describes how ctrlcclub makes decisions and manages community resources.

## Maintainers
Maintainers are responsible for:
- Setting direction for community standards and resources
- Reviewing and merging contributions
- Enforcing the Code of Conduct

## Decision-making
- We favor consensus and open discussion.
- If consensus is not reached, maintainers make the final call.
- Significant changes should be proposed in an issue before a pull request.

## Moderation
Maintainers may remove content or members who violate community standards. We prioritize safety, fairness, and clear communication.

## Updates
This governance document can be updated through the standard contribution process.
