# Contributing to ctrlcclub

Thanks for your interest in contributing. This repo hosts community resources and shared standards for ctrlcclub.

## How to contribute
1. Read the Code of Conduct and Community Guidelines.
2. Open an issue describing your proposal or improvement.
3. Submit a pull request with clear context and references.

## Contribution standards
- Keep language respectful and inclusive.
- Use clear, concise writing and short headings.
- Prefer plain text and Markdown.
- Cite sources for any borrowed ideas or text.

## Issues
Please use the issue templates to report bugs or propose ideas.

## Pull requests
- Keep changes focused and minimal.
- Explain the problem, the change, and why it helps.
- Link to a related issue when possible.

## Contributing checklist
- I read the Code of Conduct and Community Guidelines.
- I linked relevant sources, inspirations, or collaborators.
- I explained what changed and why it helps the community.
- I checked for clear, concise language and headings.
- I verified links and references.
